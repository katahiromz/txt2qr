# QRコード作成プログラム txt2qr

このプログラムは、255文字以内のテキストをQRコードに変換します。
URLを埋め込むことも可能です。

## 使い方

プログラム txt2qr.exe を起動すると、ダイアログが表示されます。
テキストを入力すれば、QRコードが表示されます。
ボタンを押せば、クリップボードにコピーしたり、
PNG画像ファイルとして保存できます。

## 動作環境

Windows Vista/7/10
